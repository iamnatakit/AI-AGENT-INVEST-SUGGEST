ขั้นตอนที่ 2: สร้าง Repository ใหม่บนเว็บ GitHub
1. เปิดเว็บไซต์ GitHub.com ล็อกอินให้เรียบร้อย
2. กดปุ่ม New (หรือเครื่องหมาย + มุมขวาบน -> New repository)
3. ตั้งชื่อ Repository เช่น investment-ai-agent-research
4. ปรับเป็น Public หรือ Private ตามต้องการ
5. ไม่ต้องติ๊ก Add a README (เพราะเรามีอยู่แล้ว)
6. กด Create repository

การ Reset ข้อมูล Log ทั้งหมด (เพื่อให้กราฟและ Report กลับไปเป็นศูนย์เริ่มต้นใหม่) สามารถทำได้ง่ายๆ โดยการลบไฟล์ฐานข้อมูลทิ้งไปเลยครับ ระบบจะสร้างไฟล์ใหม่ที่ว่างเปล่าขึ้นมาให้เองโดยอัตโนมัติ
ขั้นตอนการ Reset ล้างข้อมูล:
1. หยุดการทำงานของเซิร์ฟเวอร์ก่อน ไปที่ Terminal 1 และ Terminal 2 (ที่รัน uvicorn อยู่) แล้วกดปุ่ม Ctrl + C เพื่อหยุดรัน Backend ก่อนครับ (ถ้าไม่หยุด ระบบอาจจะล็อกไฟล์ไว้ทำให้ลบไม่ได้)
2. ลบไฟล์ฐานข้อมูลทิ้ง ไปที่โฟลเดอร์หลักของโปรเจกต์ (D:\AI AGENT INVEST SUGGEST) แล้วคลิกขวาลบไฟล์ชื่อ investment_chatbot.db ทิ้งไปเลยครับ (หรือรันคำสั่ง Remove-Item investment_chatbot.db ใน Terminal ก็ได้)
3. รันเซิร์ฟเวอร์ใหม่อีกครั้ง เมื่อคุณสั่งรัน Backend ขึ้นมาใหม่ (uvicorn project_1... และ uvicorn project_2...) ตัวโปรแกรมจะตรวจจับว่าไม่มีไฟล์ฐานข้อมูลอยู่ มันจึงจะไปดึงโครงสร้างจาก schema.sql มาสร้างตารางให้ใหม่ทั้งหมด แบบสะอาด 100% เลยครับ!
ทีนี้พอกดปุ่ม View Report บนหน้าเว็บ ตัวเลขสถิติทุกอย่างก็จะเริ่มต้นใหม่ตั้งแต่ 0 แล้วครับ!
